About

Welcome to my responsive website. I coded this project for a CSS layout course that focused on developing media-rich websites that are fully responsive and retain correct layout even on zoom.

  • Fully responsive. Page maintains proportions on zoom
  • Sticky navigation bar that fades in on scroll
  • "Back to top" button that fades in on scroll
  • Responsive image Bootstrap grid
  • Videos in MP4, Ogv and Webm formats
  • Iframe with Google Map
  • JQuery carousal plugin
  • Favicon
